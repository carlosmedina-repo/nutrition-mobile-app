## üì¶ Installation

1. **Clone the Repository**
```bash
git clone git@github.com:denizyesilirmak/nutrition-mobile.git
```

2. **Install Dependencies**
```bash
cd nutrition-mobile
npm install
```

3. **Start Development**
```bash
# Start the development server
npm start

# Run on iOS
npx run:ios --device

# Run on Android
npx run:android --device
```
